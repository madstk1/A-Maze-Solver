# A* Maze Solver

A simple A* Maze Solver written in C++. It uses a simple text file as input.


Symbols:
`S: Start node.`
`E: End node.`
`W: Wall.`
`O: Open.`

The resulting output will replace the taken path with 'X'.

## Build

To build:

```$ make```


To run with the sample file:

```$ ./astar samples/maze.txt```
